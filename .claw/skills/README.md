# Claw Skills

Ця директорія призначена для зберігання скілів, які використовуються Claw для різних робочих процесів, інструментів (MCP), та тимчасових завдань конкретного проекту.

## Структура:
- `project_specific/`: Скіли, створені тимчасово під конкретні завдання цього проекту (наприклад, iOS Remote Client, рефакторинг певного модуля).
- `workflows/`: Загальні робочі процеси та промпти (наприклад, Task Graph Roadmapping).
- `system_mcp/`: Скіли, що описують роботу з підключеними MCP (Model Context Protocol).

Щоб додати скіл до сесії, використовуйте прапорець `--attach-skill` при запуску Claw:
`claw prompt --attach-skill .claw/skills/workflows/task_graph_prompt.md "Твій запит"`
